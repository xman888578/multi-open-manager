# 🍀 在线编译 APK - 三种方案

## 方案一：GitHub Actions（最简单）⭐⭐⭐

### 步骤

#### 1. 创建 GitHub 仓库
```
1. 打开 https://github.com
2. 登录账号（没有就注册一个）
3. 点击 "+" → "New repository"
4. 仓库名：multi-open-manager
5. 选择 "Public"（公开）
6. 点击 "Create repository"
```

#### 2. 上传代码
```bash
cd /home/admin/.openclaw/workspace/multi-open-manager

# 初始化 git（如果还没有）
git init
git add .
git commit -m "Initial commit"

# 关联 GitHub 仓库（替换 YOUR_USERNAME 为你的 GitHub 用户名）
git remote add origin https://github.com/YOUR_USERNAME/multi-open-manager.git
git branch -M main
git push -u origin main
```

或者使用图形界面：
```
1. 打开 https://github.com/YOUR_USERNAME/multi-open-manager
2. 点击 "uploading an existing file"
3. 把整个 multi-open-manager 文件夹拖进去
4. 点击 "Commit changes"
```

#### 3. 等待自动编译
```
1. 点击仓库的 "Actions" 标签
2. 看到 "Build APK" 正在运行（绿色对勾表示成功）
3. 等待 2-5 分钟
```

#### 4. 下载 APK
```
1. 点击最近的构建记录（绿色对勾）
2. 滚动到底部找到 "Artifacts"
3. 点击 "multi-open-manager-debug"
4. 下载完成后解压，得到 app-debug.apk
```

## 方案二：Codemagic（无需 GitHub）⭐⭐

### 步骤

1. **打开 Codemagic**
   - 网址：https://codemagic.io
   - 注册/登录账号

2. **添加项目**
   - 点击 "Add application"
   - 选择 "Android"
   - 上传项目 ZIP 文件或连接 Git 仓库

3. **配置构建**
   - Build command: `./gradlew assembleDebug`
   - Build outputs: `app/build/outputs/apk/debug/app-debug.apk`

4. **开始构建**
   - 点击 "Start build"
   - 等待 3-8 分钟
   - 下载生成的 APK
