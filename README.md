# 🍀 多开管理器 (MultiOpen Manager)

**无会员限制 · 完全免费 · 系统级隔离**

基于 Android Work Profile（工作资料）机制的多开管理工具，无需 Root 权限。

---

## 功能特点

| 功能 | 说明 |
|------|------|
| ✅ 无限多开 | 理论上支持无限个工作空间 |
| ✅ 系统隔离 | 每个空间数据完全独立 |
| ✅ 无需 Root | 使用官方 API |
| ✅ 完全免费 | 无会员、无广告 |
| ⚠️ Android 10+ | 需要系统支持 Work Profile |

---

## 编译方法

### 环境要求
- Android Studio Hedgehog (2023.1.1) 或更高版本
- JDK 17+
- Android SDK 35

### 步骤

1. **打开项目**
   ```bash
   # 在 Android Studio 中打开 multi-open-manager 文件夹
   ```

2. **同步 Gradle**
   - File → Sync Project with Gradle Files

3. **构建 APK**
   ```bash
   # 在项目根目录执行
   ./gradlew assembleDebug
   ```

4. **输出位置**
   ```
   app/build/outputs/apk/debug/app-debug.apk
   ```

## 使用方法

### 首次设置

1. **安装 APK**
   - 将生成的 APK 传到手机
   - 允许"未知来源"安装

2. **创建工作资料**
   - 打开应用
   - 点击"创建工作资料空间"
   - 按提示完成设置

3. **切换空间**
   - 下拉通知栏可切换主空间/工作空间
   - 或在应用内直接启动

### 安装应用到工作空间

1. 切换到工作空间
2. 安装需要的应用（微信、拼多多、淘宝等）
3. 每个空间的应用互不干扰

## 常见问题

### Q: 为什么点击"创建"没反应？
A: 检查设置 → 更多设置 → 隐私 → 工作资料 是否可用。部分 ROM 可能限制了此功能。

### Q: 会被封号吗？
A: 本工具使用系统官方 API，理论上比第三方多开更安全。但任何多开行为都有被平台检测的风险，请自行评估。

### Q: 能开几个号？
A: Android 系统限制最多约 5 个工作资料，配合小米自带双开，单应用最多可开 4-5 个账号。

### Q: 数据会丢吗？
A: 删除工作资料会清空该空间所有数据。重要数据请定期备份。
