# 🍀 在线编译服务 - 详细操作指南

## 推荐服务一：Codemagic（最方便）

### 网址：https://codemagic.io

### 操作步骤

#### 1. 注册/登录
```
1. 打开 https://codemagic.io
2. 点击 "Sign in"
3. 可以用 GitHub/Google/邮箱 注册
```

#### 2. 添加应用
```
1. 登录后点击 "Add application"
2. 选择 "Android"
3. 有两个选择：
   - Connect repository（连接 GitHub）
   - Upload project（直接上传 ZIP）← 选这个
```

#### 3. 上传项目
```
1. 把 multi-open-manager 文件夹打包成 ZIP
2. 上传 ZIP 文件
3. 配置项目类型：Android
```

#### 4. 配置构建
```
Build command:
./gradlew assembleDebug

APK 输出路径:
app/build/outputs/apk/debug/app-debug.apk
```

#### 5. 开始编译
```
1. 点击 "Start build"
2. 等待 3-8 分钟
3. 下载生成的 APK
```

### 免费额度
- ✅ 500 分钟/月
- ✅ 不需要信用卡
- ✅ 注册就送

## 推荐服务二：Bitrise

### 网址：https://bitrise.io

### 操作步骤

#### 1. 注册
```
1. 打开 https://bitrise.io
2. 点击 "Sign up"
3. 用 GitHub/Google/邮箱注册
```

#### 2. 创建项目
```
1. 点击 "Dashboard" → "Create new app"
2. 选择 "I don't want to use Git"
3. 上传项目 ZIP
```

#### 3. 配置 Workflow
```
1. 选择 Android 模板
2. 添加 Gradle 步骤
3. Task: assembleDebug
```

#### 4. 编译
```
1. 点击 "Start build"
2. 等待完成
3. 下载 APK
```

### 免费额度
- ✅ 600 信用/月（约 10-12 次编译）
- ✅ 不需要信用卡

## 推荐服务三：GitHub Actions + 临时仓库（最快）

### 步骤

#### 1. 注册 GitHub
```
1. 打开 https://github.com
2. 点击 "Sign up"
3. 用邮箱注册（30 秒搞定）
```

#### 2. 创建仓库
```
1. 点击右上角头像 → "Your repositories"
2. 点击 "New"
3. 仓库名：multi-open-temp
4. 选择 "Public"
5. 点击 "Create repository"
```

#### 3. 上传代码
```
方法 A：网页上传（最简单）
1. 点击 "uploading an existing file"
2. 把整个 multi-open-manager 文件夹拖进去
3. 点击 "Commit changes"

方法 B：命令行上传
cd /home/admin/.openclaw/workspace/multi-open-manager
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/multi-open-temp.git
git push -u origin main
```

#### 4. 触发编译
```
1. 点击 "Actions" 标签
2. 看到 "Build APK" 工作流
3. 点击 "Run workflow"
4. 选择 main 分支，点击 "Run workflow"
```

#### 5. 下载 APK
```
1. 点击运行记录（绿色对勾表示完成）
2. 滚动到页面底部
3. 找到 "Artifacts"
4. 点击 "multi-open-manager-debug"
5. 等待下载完成
6. 解压得到 app-debug.apk（约 3-5MB）
```

### 免费额度
- ✅ 2000 分钟/月（个人账户）
- ✅ 不需要信用卡
