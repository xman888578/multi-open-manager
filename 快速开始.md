# 🍀 快速开始指南

## 方案一：Android Studio（推荐）⭐

### 1. 下载安装
```
下载地址：https://developer.android.com/studio
选择：Windows/Mac 版本
```

### 2. 打开项目
- 启动 Android Studio
- File → Open → 选择 `multi-open-manager` 文件夹
- 等待 Gradle 同步完成

### 3. 编译 APK
```
Build → Build Bundle(s) / APK(s) → Build APK(s)
```

### 4. 获取 APK
编译完成后，点击 "locate" 找到 APK 文件：
```
app/build/outputs/apk/debug/app-debug.apk
```

### 5. 传到手机
- 微信发给自己 → 下载 → 安装
- 或用数据线传到手机

---

## 方案二：命令行编译

### 前提条件
- 已安装 JDK 17+
- 已安装 Android SDK
- 已配置 ANDROID_HOME 环境变量

### 编译命令
```bash
cd multi-open-manager
./gradlew assembleDebug
```

### 输出
```
app/build/outputs/apk/debug/app-debug.apk
```

## 安装到小米 17 Pro Max

### 1. 允许未知来源
```
设置 → 隐私保护 → 特殊权限设置 → 安装未知应用
→ 选择你用来传 APK 的应用（如微信/文件管理）→ 允许
```

### 2. 安装 APK
- 找到 APK 文件
- 点击安装
- 同意权限

### 3. 首次使用
1. 打开 "多开管理器"
2. 点击 "创建工作资料空间"
3. 按提示完成设置
4. 下拉通知栏可切换空间

## 配合小米自带双开

**微信 4 开方案：**
```
主空间      → 微信 #1
小米双开    → 微信 #2
工作资料    → 微信 #3
工作资料双开 → 微信 #4（如果支持）
```

**切换方式：**
- 主空间/工作资料：下拉通知栏切换
- 小米双开：直接在桌面有两个图标
